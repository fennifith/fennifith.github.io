setIcon

Type: public void

Sets the icon for this view to use. It must contain a foreground image, but the background image is optional - without it, the foreground image will not be clipped and any scaling will not be applied.

Parameter Name Description
icon the icon for this view to use

setPath

Sets a custom path for this view to use, providing that its AdaptiveIcon has a background image

Parameter Name Description

path the custom path

Type: public void

Parameter Name Description
